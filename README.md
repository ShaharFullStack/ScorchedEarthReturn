## 🎮 **Controls & Gameplay**

### 🕹️ **Tank Movement**
| Key | Action |
|:---:|:---|
| `W` / `S` | Move Forward / Backward |
| `A` / `D` | Rotate Tank Body |
| `Q` / `E` | Rotate Turret |

### 🎯 **Combat Controls**
| Key | Action |
|:---:|:---|
| `↑` / `↓` | Adjust Firing Power |
| `←` / `→` | Elevate Barrel Up/Down |
| `Space` | **FIRE!** |

### 📊 **Interface**
| Key | Action |
|:---:|:---|
| `H` | Toggle Controls Info |
| `End Turn` | Complete Your Turn |
